# ВайбКумир

[Live демо на ekaterina-miazina.github.io/vibe-kumir](https://ekaterina-miazina.github.io/vibe-kumir/)

ВайбКумир это учебный проект, созданный с помощью вайб-кодинга. Это браузерная среда для знакомства с основами алгоритмизации и программирования исполнителя Робот: здесь можно писать простые программы на языке в духе КуМир, запускать их прямо в браузере и смотреть, как робот двигается по клетчатому полю, закрашивает клетки и реагирует на стены.

За основу взята учебная система КуМир. Оригинальный репозиторий: [a-a-maly/kumir2](https://github.com/a-a-maly/kumir2).

## Что умеет проект

- Редактировать программу в браузере без серверной части.
- Запускать код на поле Робота и видеть результат сразу после выполнения.
- Показывать журнал выполнения и ошибки выполнения.
- Подсвечивать синтаксические ошибки прямо в редакторе.
- Показывать номера строк и сообщения об ошибках с указанием строки.
- Поддерживать автоотступы при вводе кода.
- Позволять редактировать обстановку: менять размер поля, ставить стены, закрашивать клетки, задавать стартовую позицию робота.
- Менять масштаб поля и цвет фона.
- Открывать встроенную документацию по поддерживаемым конструкциям языка.

## Что это за язык

В проекте используется небольшой учебный поднабор языка, похожего на язык исполнителя Робот из КуМир.

Основная структура программы такая:

```kumir
использовать Робот
алг имя_алгоритма
нач
  ... команды ...
кон
```

Поддерживаются базовые команды:

- `вверх`, `вниз`, `влево`, `вправо` перемещают робота на одну клетку.
- `закрасить` закрашивает текущую клетку.
- `если <условие> то ... все` и `если <условие> то ... иначе ... все` позволяют делать ветвления.
- `нц N раз ... кц` запускает цикл фиксированное число раз.
- `нц пока <условие> ... кц` повторяет команды, пока условие истинно.
- Можно проверять состояние мира: `справа свободно`, `слева стена`, `клетка закрашена`, `клетка чистая`.
- Поддерживаются логические связки `не`, `и`, `или`.

## Примеры программ

### Линия из закрашенных клеток

Робот закрашивает клетку, на которой стоит, и идёт вправо, пока может.

```kumir
использовать Робот
алг линия
нач
  нц 4 раз
    закрасить
    если справа свободно то
      вправо
    все
  кц
кон
```

### Движение до стены

Пример цикла с условием: робот двигается вправо, пока справа есть проход.

```kumir
использовать Робот
алг до_стены
нач
  нц пока справа свободно
    вправо
  кц
кон
```

### Ветвление по состоянию клетки

Если клетка уже закрашена, робот уходит вниз. Иначе он сначала закрашивает её.

```kumir
использовать Робот
алг проверка_клетки
нач
  если клетка закрашена то
    вниз
  иначе
    закрасить
  все
кон
```

## Возможности программы

ВайбКумир полезен как маленькая учебная лаборатория:

- можно быстро проверить простую программу без установки тяжёлой среды;
- можно вручную подготовить поле под задачу и посмотреть, как алгоритм ведёт себя в разных условиях;
- можно использовать готовые примеры и встроенную документацию как шпаргалку;
- можно наглядно разбирать ошибки: где именно сломался синтаксис, на какой строке произошла ошибка выполнения, как робот пришёл к текущему состоянию.

Это не полный клон КуМир, а компактная браузерная версия с фокусом на базовые конструкции и визуальную понятность.

## Запуск проекта

Все команды выполняются из корня репозитория.

### Требования

- Node.js 20+ желательно актуальной LTS-версии
- npm

### Установка зависимостей

```bash
npm install
```

### Локальная разработка

```bash
npm run dev
```

Приложение поднимается на адресе [http://127.0.0.1:4173/](http://127.0.0.1:4173/).

### Проверка качества

Проверка TypeScript:

```bash
npm run lint
```

Юнит-тесты:

```bash
npm test
```

End-to-end тесты:

```bash
npm run test:e2e
```

Сборка production-версии:

```bash
npm run build
```

### Стек проекта

- Vue 3
- TypeScript
- Vite
- Vitest
- Playwright
